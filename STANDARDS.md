# STANDARDS.md

## BAIS:3300 - Digital Product Management · Personal Landing Page

---

## 1. Project Overview

This is a personal landing page for Angela Maranto, a Business Analytics student at the University of Iowa. The site is designed for recruiters and hiring managers at Fortune 500 tech companies, especially in Chicago, to quickly understand her background, skills, and project experience. A successful outcome is generating interview opportunities and increasing professional visibility.

---

## 2. Technical Standards

**Languages and versions:**
- HTML5 (semantic elements required)
- CSS3 (external stylesheet only)

**Folder structure:**
/project-root  
├── index.html  
├── /css  
│    └── stylesheet.css  
├── /images  
│    └── headshot.jpg  

**Architecture:**
- Static site (HTML/CSS only)
- Single index.html file
- No backend, no database
- No JavaScript required
- Images must be stored locally in images folder

**Responsiveness:**
- Fully responsive (mobile + desktop)
- No horizontal scrolling

**Accessibility:**
- All images must have alt text
- Proper heading structure (h1 → h2 → h3)
- Descriptive links only

**Security:**
- External links open in new tab (target="_blank")

---

## 3. Design Standards

**Color palette:**
- Background: #F8F9FA
- Primary text: #212529
- Accent: #0D6E6E
- Secondary background: #E9ECEF

**Typography:**
- Font: Inter (Google Fonts)
- Body: 16px
- H1: 1.5rem bold
- H2: 1.25rem bold

**Layout:**
- Max width: 800px
- Centered layout
- Clean and minimal spacing
- Sticky navigation bar

**Components:**
- Profile photo: circular, centered
- Skills: pill-style tags
- Contact: LinkedIn, GitHub, email

**Tone:**
- Professional
- Approachable
- Data-driven
- First-person writing
